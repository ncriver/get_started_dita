Using Sign Data

Please be aware of the instructions before using the sign data.

  1. Do not publicize the sign data to control the number of visitors.
  2. Note that the GPS coordinates refer to the trailhead, not the parking area. You may have to park some distance away from the trailhead.
  3. Please update any incorrect information.
